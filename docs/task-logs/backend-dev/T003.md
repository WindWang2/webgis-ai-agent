## 技术实现

### 1. Celery 配置

```python
celery_app = Celery(
    "webgis_ai",
    broker=settings.CELERY_BROKER_URL,  # redis://localhost:6379/0
    backend=settings.CELERY_RESULT_BACKEND,
    task_time_limit=3600,  # 1 小时超时
    task_soft_time_limit=3300,  # 55 分钟软超时
)
```
