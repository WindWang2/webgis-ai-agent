## 迁移说明

### 迁移原因

按照项目规范，前端代码需要放在 `frontend/` 子目录下，以便：
1. 前后端代码清晰分离
2. 便于多模块协作开发
3. 符合 monorepo 结构规范

### 迁移内容

| 原位置 | 新位置 |
|--------|--------|
| `app/` | `frontend/app/` |
| `components/` | `frontend/components/` |
| `lib/` | `frontend/lib/` |
| `*.json` / `*.mjs` | `frontend/` |
