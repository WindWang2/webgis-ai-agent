## 任务描述

实现图层上传、列表展示、空间分析任务提交、进度查看、结果渲染到地图

### 功能需求

1. **图层上传**
   - 支持常见 GIS 格式（Shapefile、GeoJSON、KML、GeoTIFF）
   - 拖拽上传 + 文件选择
   - 上传进度显示
   - 文件格式校验

2. **图层列表展示**
   - 已上传图层列表
   - 图层缩略图
   - 图层元数据（名称、大小、创建时间、坐标系）
   - 图层开关控制
   - 图层透明度调节
   - 图层删除功能

3. **空间分析任务提交**
   - 分析类型选择（缓冲区分析、叠加分析、网络分析等）
   - 参数配置面板
   - 任务提交按钮
   - 参数校验

4. **进度查看**
   - 任务队列展示
   - 实时进度条
   - 状态更新（排队中、处理中、完成、失败）
   - 日志输出

5. **结果渲染到地图**
   - 分析结果图层自动加载
   - 结果高亮显示
   - 结果属性查询
   - 结果导出

## 技术实现

### 组件结构

```
frontend/components/
├── layer/
│   ├── layer-upload.tsx      # 图层上传组件
│   ├── layer-list.tsx        # 图层列表组件
│   ├── layer-item.tsx        # 单个图层项
│   └── layer-controls.tsx    # 图层控制（开关、透明度）
├── analysis/
│   ├── analysis-panel.tsx    # 分析配置面板
│   ├── analysis-types.tsx    # 分析类型选择
│   └── task-progress.tsx     # 任务进度展示
└── map/
    └── map-with-layers.tsx   # 集成图层管理的地图组件
```

### API 接口（待后端确认）

```typescript
// 图层相关
POST /api/layers/upload          // 上传图层
GET  /api/layers                 // 获取图层列表
GET  /api/layers/:id             // 获取图层详情
DELETE /api/layers/:id           // 删除图层

// 分析任务相关
POST /api/analysis/tasks         // 提交分析任务
GET  /api/analysis/tasks         // 获取任务列表
GET  /api/analysis/tasks/:id     // 获取任务详情
GET  /api/analysis/tasks/:id/ws  // WebSocket 进度
```
