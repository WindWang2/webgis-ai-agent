# WebGIS AI Agent 任务看板（任务驱动模式 · Superpowers标准）
## 🌟 核心开发规则（强制执行）
✅ **纯任务驱动开发**：无任何时间/日期限制，无"明天/后天/下周"等推迟表述，任务完成后立即自动启动下一优先级最高任务
✅ **优先级严格执行**：永远按优先级从高到低处理任务，禁止跳过前面的高优先级任务处理低优先级任务
✅ **单任务并行限制**：同一时间Coder只能处理1个进行中任务，完成后自动取下一个最高优先级任务
✅ **状态流转规则**：待办 → 进行中 → 已完成（若阻塞则移入阻塞队列）

## 📌 看板更新规则（Superpowers流程标准）
### 🔄 必须更新场景：
1. **任务状态变更**：任务开始/完成/阻塞时，立即更新状态
2. **PR合并**：PR合并后立即将对应任务标记为已完成，填写关联PR编号
3. **Issue关闭**：Issue关闭后立即将对应任务标记为已完成，填写关联Issue编号
4. **优先级调整**：任务优先级变更时，立即调整待办队列顺序
### 📝 更新要求：
- 所有字段必须填写完整，禁止遗漏
- 已完成任务必须填写完成时间和关联PR/Issue
- 待办队列必须严格按照优先级排序，禁止打乱顺序
