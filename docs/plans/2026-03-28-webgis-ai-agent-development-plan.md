## 开发流程要求
1. 所有功能采用TDD模式开发：先编写组件测试用例，再实现功能代码，确保测试通过
2. 每个模块开发完成后先进行本地自测，再提交API联调
3. 代码提交前必须通过ESLint、TypeScript类型检查
4. 联调完成后提交PR，经代码审核后合并
